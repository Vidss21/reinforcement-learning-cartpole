# Reinforcement Learning Agent for CartPole

This project implements a Reinforcement Learning agent using Q-learning to solve the CartPole control problem.

## Project Overview
The CartPole environment consists of a pole attached to a moving cart. The objective is to keep the pole balanced by applying forces to move the cart left or right.

The RL agent learns optimal actions through interaction with the environment using reward-based learning.

## Methodology
- Environment: OpenAI Gym CartPole-v1
- Algorithm: Q-learning
- State space discretization
- Epsilon-greedy exploration strategy

## Technologies Used
- Python
- NumPy
- Gymnasium
- Matplotlib
- Reinforcement Learning

## Results
The agent gradually improves its performance and learns to balance the pole for longer durations as training progresses.
